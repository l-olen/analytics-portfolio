# Квартальный отчёт ОМТ-Консалт: инструкция по подготовке

Отчёт выходит раз в квартал (I кв. = апрель, II кв. = июль, III кв. = октябрь, IV кв. = январь).
Охватывает Казахстан, Узбекистан, Кыргызстан, Таджикистан.
Формат -- PPTX-презентация (~12 слайдов для Казахстана, 4-6 слайдов для остальных).

---

## 1. Первичные данные: что обновлять каждый квартал

| Источник | Что содержит | Куда идёт в Excel |
|---|---|---|
| Данные о производстве КАЗ (от НПЗ / Казстат) | Ежемесячный выпуск БН и ПБВ по производителям | Таблица в файле, путь = `Путь_Производство_Каз` |
| Данные о производстве УЗБ (Ферганский НПЗ / Узстат) | Ежемесячный выпуск БН по производителям | Таблица в файле, путь = `Путь_Производство_Узб` |
| ВЭД Казахстан (Комитет государственных доходов РК) | Таможенные данные по импорту и экспорту | Таблица `Таблица_ВЭД_Каз` в файле `Путь_ВЭД_Каз` |
| ВЭД Узбекистан (Государственный таможенный комитет РУз) | То же для Узбекистана | `Путь_ВЭД_Узб` |
| ВЭД Кыргызстан + Таджикистан | Импорт/экспорт БН и ПБВ | `Путь_ВЭД_КыргТадж` |
| Цены Казахстан | Индикативная ежемесячная цена БНД, тг/т | Таблица `КАЗ_по_месяцам` в файле `Путь_ЦеныКАЗ` |
| Цены Узбекистан (биржа) | Мин/макс/средняя ежемесячная цена в сум/т | Таблица `БНД_по_месяцам` в файле `Путь_УзБиржа` |
| Цены РФ | Для сравнения с казахстанскими ценами | `Путь_ЦеныРФ` |
| Курсы валют | Для пересчёта российских цен в тенге/сум | `Путь_КурсыВалют` (файл КурсыВалют.xlsx) |

**Важно:** параметры `Путь_*` задаются в Excel-файле на листе с настройками Power Query.
Если путь к исходному файлу изменился -- обновить соответствующий параметр.

---

## 2. Файл Сборка_квартальные.xlsx: структура

Главный файл -- `ОМТ/Квартальный отчет/Сборка_квартальные.xlsx`.

### Листы с данными (графики):
- `Казахстан` -- графики по Казахстану (данные тянутся через фильтр из ИТОГ_* таблиц)
- `Казахстан ПБВ` -- аналогично для ПБВ
- `Узбекистан` -- данные по производству и ценам Узбекистана
- `Кыргызстан`, `Таджикистан` -- импорт/экспорт для этих стран

### Параметры (Power Query):
- `ДатаОтчета` -- дата последнего месяца отчётного периода (например, 31.03.2026 для I квартала)
- `МесяцОтчета` -- номер последнего месяца (1 для января, 3 для марта и т.д.)
- `Путь_*` -- пути к файлам-источникам

**Каждый квартал обновлять**: `ДатаОтчета` и `МесяцОтчета`.

## 3. Power Query запросы: описание

### Исходные запросы (читают первичные данные)

| Запрос | Источник | Описание |
|---|---|---|
| `Производство_Каз` | Файл по пути `Путь_Производство_Каз` | Ежемесячные данные о выпуске БН/ПБВ по заводам Казахстана. Столбцы: Год, Дата, Вид, Компания, Производитель, Кластер, Регион, Объём тыс.т |
| `Производство_Узб` | `Путь_Производство_Узб` | То же для Узбекистана |
| `Производство_Отгрузки_Каз` | Объединение Производство + Отгрузки | Добавляет Страна="Казахстан", Показатель="Производство" или "Отгрузки". Используется во всех ИТОГ_Произв_* и ИТОГ_Отгрузки_Каз |
| `ВЭД_Каз` | `Путь_ВЭД_Каз` | Таможенные данные по импорту/экспорту КАЗ: Дата, СтранаОтчета, Направление, СтранаОтправления, СтранаНазначения, Компания, НПЗ, Вид, КодТНВЭД, Объём тыс.т |
| `ВЭД_Узб` | `Путь_ВЭД_Узб` | То же для Узбекистана, нормализация текста |
| `ВЭД_КыргТадж` | `Путь_ВЭД_КыргТадж` | Кыргызстан + Таджикистан |
| `ВЭД` | Объединение ВЭД_Каз + ВЭД_Узб + ВЭД_КыргТадж | Единый справочник для всех ИТОГ_Имп_* и ИТОГ_Экс_* |
| `ЦЕНЫ_Каз_Узб` | `Путь_ЦеныКАЗ`, `Путь_УзБиржа`, `Путь_ЦеныРФ`, `Путь_КурсыВалют` | Цены в тыс.тг (Казахстан + РФ) и млн.сум (Узбекистан + РФ) |

### Итоговые запросы (данные для графиков)

Все итоговые таблицы выгружаются на отдельные листы и используются Python-скриптом через xlwings.

| Запрос | Страны | Период | Столбцы | Назначение |
|---|---|---|---|---|
| `ИТОГ_Произв_Год` | КАЗ + УЗБ | 5 лет | Страна, Вид, Год, Объем_Год, Объем_Период, Динамика_% | Слайд "Производство -- годовая динамика" |
| `ИТОГ_Произв_Месяц` | КАЗ + УЗБ | 3 года | Страна, Вид, НомерМесяца, [год1], [год2], [год3] | Помесячный график производства |
| `ИТОГ_Произв_Производитель` | КАЗ + УЗБ | 2 года | Страна, Вид, Производитель, Год, Объем_тыс_т, Доля_% | Структура по заводам |
| `ИТОГ_Отгрузки_Каз` | КАЗ | Период | Кластер/Регион, Год, данные | Региональная структура отгрузок |
| `ИТОГ_Имп_Год` | Все 4 | 5 лет | Страна, Вид, Год, Объем_Год, Объем_Период, Динамика_% | Слайд импорта -- годовая динамика |
| `ИТОГ_Имп_Мес` | Все 4 | 3 года | Страна, Вид, НомерМесяца, [год1], [год2], [год3] | Помесячная динамика импорта |
| `ИТОГ_Имп_Страна` | Все 4 | 2 года | Страна, Вид, СтранаОтправления, Год, Доля_% | Поставщики: Россия, другие |
| `ИТОГ_Экс_Год` | Все 4 | 5 лет | Страна, Вид, Год, Объем_Год, Объем_Период, Динамика_% | Слайд экспорта -- годовая динамика |
| `ИТОГ_Экс_Мес` | Все 4 | 3 года | Страна, Вид, НомерМесяца, [год1], [год2], [год3] | Помесячная динамика экспорта |
| `ИТОГ_Экс_Направл` | Все 4 | 2 года | Страна, Вид, СтранаНазначения, Год, Доля_% | Направления экспорта |
| `ИТОГ_Баланс` | КАЗ | Период | Производство, Импорт, Экспорт, Потребление | Битумный баланс Казахстана |
| `ИТОГ_Цены_КазУзб` | КАЗ + УЗБ + РФ | 3 года | Страна, Год, НомерМесяца, ТипЦены, ЕдИзм, Цена | Слайд цен (отдельно тг и сум) |

**Фильтр для графиков цен:**
- `ЕдИзм = "тыс. тг"` -- Казахстан + Россия (для казахстанского слайда)
- `ЕдИзм = "млн. сум"` -- Узбекистан + Россия (для узбекского слайда)

### Известная ошибка в PQ (исправить вручную в трёх запросах)

Запросы `ИТОГ_Произв_Год`, `ИТОГ_Имп_Год`, `ИТОГ_Экс_Год` содержат self-join без `Table.Buffer`.
Из-за ленивой оценки PQ все значения `Динамика_%` получаются null или неверными.

Шаги исправления (в Advanced Editor для каждого из трёх запросов):
1. Найти строку `Сорт = Table.Sort(Merged, ...)` -- переименовать в `СортРаw`
2. Добавить: `Сорт = Table.Buffer(СортРаw),`
3. Убедиться, что self-join (`Joined = Table.NestedJoin(...)`) использует `Сорт` на обеих сторонах
4. Добавить финальную сортировку в `Итог`

Подробный код исправления -- в файле `ОМТ/pq_итог_год_fix.txt`.

## 4. Пошаговый процесс подготовки отчёта

### Шаг 1. Получить первичные данные
- Обновить файлы-источники (производство, ВЭД, цены) за отчётный квартал
- Если файл переименован или переложен -- обновить параметр `Путь_*` в Excel

### Шаг 2. Обновить параметры в Excel
Открыть `Сборка_квартальные.xlsx` → Данные → Запросы и подключения:
- Параметр `ДатаОтчета` → последний день отчётного месяца (например, 31.03.2026)
- Параметр `МесяцОтчета` → номер последнего месяца (например, 3 для марта)

### Шаг 3. Обновить все запросы Power Query
Данные → Обновить всё (или Ctrl+Alt+F5). Дождаться завершения.

После обновления проверить данные на листах:
- `ИТОГ_Произв_Год`: убедиться, что `Динамика_%` не пустая (не null)
- `ИТОГ_Имп_Год`, `ИТОГ_Экс_Год`: то же

### Шаг 4. Обновить analyst_context_current.md
Файл `ОМТ/Квартальный отчет/analyst_context_current.md` -- контекст рынка для Claude.
Раз в квартал обновлять вручную:
- Актуализировать цены нефти и курсы валют
- Добавить значимые события квартала (регуляторные решения, производственные изменения)
- Обновить квотные условия, прогнозы, новости по производителям

Дополнительный источник событий -- `ОМТ/Еженедельный отчёт/weekly_events_log.md` (заполняется автоматически из еженедельного скрипта).

### Шаг 5. Сгенерировать аналитические тексты
```powershell
cd "C:\projects\my-project\ОМТ\Квартальный отчет"
python generate_quarterly_slides.py kaz     # Казахстан
python generate_quarterly_slides.py uzb     # Узбекистан
python generate_quarterly_slides.py kgz     # Кыргызстан
python generate_quarterly_slides.py tjk     # Таджикистан
```

Результат: файлы `analytics_slides_{страна}_{дата}.txt` -- тексты по структуре слайдов.
Каждый раздел содержит:
- `ПОДЗАГОЛОВОК` -- одна строка, главная тенденция (вставляется в PPTX как подпись слайда)
- `ТЕКСТ` -- 2-4 абзаца аналитики с цифрами

### Шаг 6. Обновить PPTX
- Открыть файл отчёта предыдущего квартала
- Обновить графики (они связаны с Excel через OLE или тянут данные с листов)
- Вставить подзаголовки и тексты из `.txt` файла
- Обновить заголовок и дату на титульном слайде

## 6. Ключевые особенности и подводные камни

**Казахстан -- нарратив квоты на импорт:**
Снижение квоты с 500 до 50 тыс.т в 2026 г. -- это решение самого Казахстана (промышленный протекционизм), а не ограничение со стороны РФ. Но есть рыночное противоречие: Caspi Bitum в Атырау, логистика в центральные/южные регионы дороже российского импорта -- отсюда проблемы с заявками у казахстанских производителей. При написании аналитики важно отражать оба аспекта: политику и рыночную реальность.

**Excel + Power Query:**
Использовать только xlwings (не openpyxl). openpyxl ломает Power Query.

**Производство:**
- Данные по производству КАЗ -- от ПНХЗ, Qazaq Bitum, Caspi Bitum, Асфальтобетон 1
- Данные по производству УЗБ -- от Ферганского НПЗ (единственный производитель)
- Кыргызстан и Таджикистан производства нет (только импорт)

**Экспорт Казахстана:**
- БН экспортируется только в Кыргызстан (единственный экспортёр -- Qazaq Bitum, регулируется государством)
- ПБВ -- первые поставки в Узбекистан появились в 2026 г.
- С 01.07.2026 действует запрет на экспорт БН (код ТН ВЭД 2713) на полгода

**Импорт ПБВ в Казахстан:**
ПБВ идёт по коду ТН ВЭД 2715 -- не подпадает под квоту на базовый битум (2713). В 2026 г. импорт ПБВ резко вырос на фоне сжатия импорта БН.
